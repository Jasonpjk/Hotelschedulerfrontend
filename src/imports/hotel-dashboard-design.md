방금 완성한 로그인 페이지와 동일한 톤앤매너를 유지해서,
로그인 후 진입하는 호텔 프런트 운영 스케줄 시스템의 메인 대시보드 페이지를 디자인해줘.

중요 연결 조건:
- 로그인 페이지에서 선택한 언어 설정이 이 대시보드에도 그대로 이어지는 구조처럼 보여야 해
- 즉, 로그인 화면의 언어 선택값이 유지된 상태로 대시보드에 진입한 것처럼 설계해줘
- 현재 기본 언어는 한국어로 보여주되, 영어, 일본어, 러시아어, 베트남어로 전환 가능한 구조를 유지해줘
- 상단 헤더 우측에 언어 선택 UI를 유지하거나 자연스럽게 연결해줘
- 언어가 바뀌어도 카드, 버튼, 메뉴, 표 제목 길이가 늘어나도 레이아웃이 깨지지 않게 설계해줘

공통 톤앤매너:
- 일반 소비자용 예약 사이트가 아니라 호텔 운영 관리자용 SaaS 대시보드
- 로그인 페이지와 동일한 프리미엄 호텔 브랜드 무드
- 롯데호텔 홈페이지가 주는 고급스럽고 정제된 느낌 참고
- 딥 네이비, 차콜, 웜 화이트, 절제된 골드 포인트 유지
- 절제된 럭셔리, 신뢰감, 넓은 여백, 정돈된 그리드, 얇은 선
- 지나치게 화려한 차트나 마케팅 배너 금지
- 스타트업 느낌의 가벼운 대시보드가 아니라 품격 있는 운영 관리자 툴 느낌
- 한글과 외국어 모두 고급스럽고 정제된 프리미엄 호텔 브랜드 무드의 타이포그래피로 보여줘
- 정확히 특정 폰트를 복제하기보다, 유사한 품격과 분위기를 유지하는 방향으로 설계해줘

프로젝트 맥락:
- 이 서비스는 호텔 프런트 직원 스케줄을 생성, 조회, 수정, 확정하는 운영 시스템
- 대시보드는 로그인 후 처음 보는 허브 화면
- 가장 중요한 핵심 기능은 월별 근무표 메인 페이지로 이동하는 것
- 관리자 입장에서 오늘 확인할 상태와 바로 해야 할 액션이 한눈에 보여야 함

필수 구성:
1. 상단 헤더
- 브랜드/서비스명
- 페이지 타이틀: 대시보드
- 호텔 선택 드롭다운
- 언어 선택 UI
- 알림 아이콘
- 사용자 프로필 / 계정 메뉴

2. 좌측 사이드바
- 대시보드
- 근무표
- 직원 관리
- 휴무 관리
- 수요 입력
- 호텔 설정
- 현재 대시보드 메뉴 선택 상태가 분명히 보여야 함

3. 상단 KPI 카드
- 이번 달 스케줄 상태
- 미확정 항목 수
- 락 예정 항목 수
- 전체 직원 수
- 리스크 날짜 수

4. 메인 핵심 카드
- 이번 달 스케줄 상태 카드
  - 현재 월
  - 현재 버전
  - 상태 배지(초안, 확정, 락)
  - 마지막 수정 시간
  - 버튼: 근무표 보기, 새로 생성, 다운로드, 확정
- 다음 달 생성 바로가기 카드
  - 다음 달 준비 상태
  - 생성 시작 버튼
  - 최근 생성 결과 요약

5. 보조 카드 영역
- 공정성 요약
- 최근 변경 이력
- AI 조정 기록
- 운영 경고 / 리스크

6. 빠른 액션 영역
- 직원 추가
- 휴무 등록
- 수요 입력
- 근무표 바로 열기
- 호텔 설정 열기

- 새로 생성
- 다운로드
- 확정

국제화/다국어 조건:
- 현재 시안은 한국어 기준으로 보여줘
- 하지만 영어, 일본어, 러시아어, 베트남어로 바뀌어도 같은 톤앤매너와 가독성이 유지되게 설계해줘
- 언어 전환 후에도 버튼 크기, 카드 폭, 메뉴 너비, 헤더 구조가 안정적으로 유지되게 해줘
- 다국어 지원 SaaS 관리자 화면처럼 보이게 해줘

최종 목표:
- 로그인 페이지와 완전히 연결된 디자인 시스템
- 로그인에서 선택한 언어가 그대로 이어지는 것처럼 보이는 관리자 대시보드
- 프리미엄 호텔 브랜드 무드와 실제 운영 효율이 동시에 느껴지는 메인 허브 화면
