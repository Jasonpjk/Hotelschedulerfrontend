현재 LOTTE HOTELS & RESORTS 프런트 스케줄 SaaS의 “스케줄 재생성” 화면에서
야간조 조정 옵션이 제대로 작동하지 않는 문제를 수정해줘.

현재 화면에서는 “야간조 계획도 함께 조정”을 체크하고,
야간조 조정 방식에서 “직접 우선 대상자 선택” 또는 “직접 제외 대상자 선택”을 선택해도
실제로 누구를 우선 대상자로 넣고 누구를 제외할지 선택하는 UI가 나타나지 않고
그냥 다음 단계로 넘어가고 있어.

이번 작업의 핵심은
야간조 조정 방식을 선택했을 때 실제 선택 UI가 반드시 열리고,
선택한 우선 대상자 / 제외 대상자가 스케줄 재생성 결과와 연간 야간조 계획에 실제로 반영되도록 만드는 것이야.

--------------------------------------------------
[현재 문제]
--------------------------------------------------

현재 스케줄 재생성 화면의 야간조 조정 영역에는 아래 옵션이 있음.

- 야간조 계획도 함께 조정
- 자동 재설정
- 직접 우선 대상자 선택
- 직접 제외 대상자 선택

하지만 문제는 다음과 같음.

1. “직접 우선 대상자 선택”을 눌러도 직원 선택 UI가 열리지 않음
2. “직접 제외 대상자 선택”을 눌러도 직원 선택 UI가 열리지 않음
3. 우선 대상자와 제외 대상자를 선택하지 않았는데도 다음 단계로 넘어감
4. 선택한 대상자가 없는데도 스케줄 재생성 실행이 가능해 보임
5. 연간 야간조 계획에 실제 반영되는 구조가 보이지 않음

--------------------------------------------------
[수정 목표]
--------------------------------------------------

야간조 조정 방식별로 동작을 명확히 구분해줘.

1) 자동 재설정
- 별도 직원 선택 UI 없이 시스템이 운영정책 기준으로 자동 재설정
- 단, 결과 미리보기에서 어떤 직원이 우선 대상자 / 제외 대상자로 반영되는지 보여줘

2) 직접 우선 대상자 선택
- 직원 선택 UI가 반드시 열려야 함
- 스케줄러가 이번 재생성에서 야간조 우선 대상자로 넣을 직원을 직접 선택할 수 있어야 함
- 선택한 직원은 해당 월 연간 야간조 계획의 “우선 대상자”에 반영되어야 함

3) 직접 제외 대상자 선택
- 직원 선택 UI가 반드시 열려야 함
- 스케줄러가 이번 재생성에서 야간조 제외 대상자로 뺄 직원을 직접 선택할 수 있어야 함
- 선택한 직원은 해당 월 연간 야간조 계획의 “제외 대상자”에 반영되어야 함

4) 직접 우선 + 직접 제외를 동시에 설정할 수 있어야 함
- 우선 대상자만 선택 가능
- 제외 대상자만 선택 가능
- 우선 대상자와 제외 대상자 둘 다 선택 가능

--------------------------------------------------
[야간조 조정 UI 구조]
--------------------------------------------------

“야간조 계획도 함께 조정” 체크박스를 켜면
아래 영역이 펼쳐지게 해줘.

제목:
야간조 조정 설정

설명:
“이번 스케줄 재생성 결과를 해당 월 연간 야간조 계획에도 반영합니다.”

선택 방식:
- 자동 재설정
- 직접 선택

여기서 “직접 선택”을 선택하면 아래 두 개의 선택 영역이 나타나야 함.

1. 우선 대상자 선택
2. 제외 대상자 선택

각 영역에는 직원 목록이 표시되어야 함.

--------------------------------------------------
[직접 선택 UI]
--------------------------------------------------

직접 선택 UI는 아래처럼 구성해줘.

[우선 대상자 선택]
- 직원 검색
- 직급 필터
- 현재 조 필터
- 직원 목록
- 다중 선택 가능
- 선택된 인원 수 표시

[제외 대상자 선택]
- 직원 검색
- 직급 필터
- 현재 조 필터
- 직원 목록
- 다중 선택 가능
- 선택된 인원 수 표시

표시 정보:
- 직원명
- 직급
- 역할
- 현재 주 배치 그룹
- 인차지 여부
- 해당 월 기존 야간조 여부
- 보호 일정 여부
- 장기 병가 / 퇴사 / 교육 / 신혼여행 등 제외 사유가 있으면 표시

중요:
- 대체 후보를 별도로 시스템이 제한해서 보여줄 필요는 없음
- 전체 직원 중에서 선택 가능하게 하되, 선택 불가 사유가 있는 직원은 비활성 또는 경고 표시
- 스케줄러가 판단해서 직접 선택할 수 있어야 함

--------------------------------------------------
[직접 선택 시 검증 규칙]
--------------------------------------------------

직접 선택 시 아래 검증을 적용해줘.

1) 같은 직원을 우선 대상자와 제외 대상자에 동시에 넣을 수 없음
- 만약 우선 대상자로 선택된 직원을 제외 대상자로 선택하려 하면 확인창 표시

확인창:
“이 직원은 이미 우선 대상자로 선택되어 있습니다.
제외 대상자로 변경하시겠습니까?”

버튼:
- 제외 대상자로 변경
- 우선 대상자 유지
- 취소

2) 절대 보호 기간이 있는 직원은 야간조 우선 대상자로 선택 시 경고 표시
예:
“이 직원은 해당 기간에 절대 보호 일정이 있습니다. 야간조 배정이 제한될 수 있습니다.”

3) 퇴사 이후 기간이 포함된 직원은 우선 대상자로 선택 불가
4) 장기 병가 기간이 포함된 직원은 우선 대상자로 선택 불가
5) 승인된 교육/병가/휴일 안내 일정과 충돌하면 결과 미리보기에서 경고 표시

--------------------------------------------------
[야간조 운영 원칙 반영]
--------------------------------------------------

야간조 조정을 실행할 때 아래 원칙을 유지해줘.

- 야간조 대상자는 해당 월에 N22 중심으로 배치
- 야간조 대상자는 같은 월에 M07 오전조와 섞이지 않음
- 야간조 대상자는 같은 월에 A13 오후조와 섞이지 않음
- 야간조 대상자는 같은 월에 C08 / C09 / C10 / C11 중간조와 섞이지 않음
- OFF / 휴식일은 가능
- 변경 기준일 이전 일정은 유지
- 절대 보호 일정은 변경 금지
- 우선 보호 일정은 가능하면 유지

--------------------------------------------------
[스케줄 재생성 실행 전 필수 체크]
--------------------------------------------------

야간조 조정 옵션이 켜져 있고 조정 방식이 “직접 선택”인 경우,
아래 중 하나라도 설정되어 있어야 스케줄 재생성 버튼이 활성화되게 해줘.

- 우선 대상자 1명 이상 선택
또는
- 제외 대상자 1명 이상 선택

아무도 선택하지 않은 상태에서 “스케줄 재생성”을 누르면 아래 안내를 표시해줘.

toast 또는 inline warning:
“직접 선택 방식을 사용하려면 우선 대상자 또는 제외 대상자를 1명 이상 선택해주세요.”

단, “자동 재설정” 방식은 직원 선택 없이 실행 가능.

--------------------------------------------------
[결과 미리보기 반영]
--------------------------------------------------

스케줄 재생성 후 결과 미리보기에는 야간조 조정 결과가 반드시 표시되어야 함.

표시 항목:
- 야간조 조정 여부
- 조정 방식: 자동 재설정 / 직접 선택
- 우선 대상자 변경 내역
- 제외 대상자 변경 내역
- 연간 야간조 계획 변경 내역
- 월별 근무표 N22 배정 변경 내역
- 야간조 원칙 위반 여부
- 남은 경고

예시:
야간조 계획 변경 있음

우선 대상자:
- 추가: 이서연, 김민준
- 유지: 최유진

제외 대상자:
- 추가: 박지호
- 사유: 장기 병가

월별 근무표 변경:
- 박지호 05-12 N22 → SICK
- 이서연 05-12 OFF → N22
- 이서연 05-13 OFF → N22

--------------------------------------------------
[새 버전 저장 시 실제 반영]
--------------------------------------------------

“새 버전으로 저장”을 누르면 아래에 실제 반영되어야 함.

1) 월별 근무표
- N22 배정 변경
- 야간조 제외 대상자 반영
- 보호 일정 유지
- 변경 기준일 이후만 반영

2) 연간 야간조 계획
- 해당 월 우선 대상자 갱신
- 해당 월 제외 대상자 갱신
- 예비 대상자는 사용하지 않음

3) 변경 이력
- 야간조 조정 내역 기록
- 누가 우선 대상자로 추가되었는지 기록
- 누가 제외 대상자로 추가되었는지 기록
- 생성된 버전 기록

4) 대시보드 / 운영 경고 / 공정성 리포트
- 야간조 편중
- 야간조 부족
- 야간조 원칙 위반 여부 갱신

--------------------------------------------------
[연간 야간조 계획 구조]
--------------------------------------------------

연간 야간조 계획은 아래 두 그룹만 사용해줘.

- 우선 대상자
- 제외 대상자

예비 대상자는 사용하지 마.

스케줄 재생성에서 야간조 조정이 발생하면
해당 월 연간 야간조 계획 카드에도 즉시 반영되어야 함.

예:
변경 전:
5월 우선 대상자: 박지호, 최유진
5월 제외 대상자: 없음

변경 후:
5월 우선 대상자: 최유진, 이서연
5월 제외 대상자: 박지호

--------------------------------------------------
[UI 표시 개선]
--------------------------------------------------

현재 화면에서는 “직접 우선 대상자 선택”을 선택해도 아무 UI가 안 떠서 사용자가 진행 상태를 알 수 없음.

수정 후에는 아래처럼 보여야 함.

예:
야간조 계획도 함께 조정: ON

조정 방식:
● 자동 재설정
○ 직접 선택

직접 선택 선택 시:
[우선 대상자 선택] 0명 선택
[직원 선택하기]

[제외 대상자 선택] 0명 선택
[직원 선택하기]

또는 직원 목록을 바로 펼쳐서 선택 가능하게 해도 됨.

직접 우선 대상자 선택 / 직접 제외 대상자 선택을 별도 radio로 둘 경우:
- 선택 즉시 해당 직원 선택 영역이 펼쳐져야 함
- 선택 영역이 안 뜨면 안 됨
