- 작업 월 드롭다운 추가

중요:
- 제목은 “월별 수요 예측”
- 실제 작업 대상은 “작업 월” 드롭다운으로 선택
- 사용자가 어떤 월의 데이터를 보고 있는지 헷갈리지 않게 정리

필수 반영:

예시:
- 1개월 전
- 2개월 전
- 3개월 전
...
- 12개월 전

또는

- 이력 보기

- “예측근거” 탭에서는
  - 다운로드 버튼 비활성화
  - 이력 보기 버튼 비활성화

비활성 상태 조건:
- 클릭 불가
- hover 강조 없음
- 흐린 회색 또는 반투명 회색 계열
- 포인터 커서로 보이지 않게

  결과 다운로드나 이력 보기 액션이 활성화되면 안 됨

반드시 반영:

- 이력 보기 버튼 활성화

활성 상태 조건:
- 클릭 가능
- hover 시 인터랙션 보임
- 포인터 커서 적용
- 실제 결과 작업 버튼처럼 보이게

즉,
탭 상태에 따라 아래처럼 명확히 구분되어야 합니다.
- 예측근거: 다운로드 / 이력 보기 비활성
- 예측결과: 다운로드 / 이력 보기 활성
