현재 LOTTE HOTELS & RESORTS 관리자용 SaaS의 “직원 관리” 화면을 아래 기준으로 수정해주세요.

중요:
- 이번 수정은 “직원 관리” 화면에만 적용해주세요.
- 다른 화면은 건드리지 말아주세요.
- 기존 시스템의 프리미엄 호텔 운영 관리자용 SaaS 톤앤매너 유지
- 딥 네이비 / 웜 화이트 / 절제된 골드 포인트 / 넓은 여백 / 얇은 선 / 정돈된 그리드 유지

[1. 오른쪽 액션 버튼명 변경]
현재 테이블 제일 오른쪽 “관리” 컬럼 아래에 있는 버튼명을
“관리”에서 “설정”으로 변경해주세요.

변경 전:
- 관리

변경 후:
- 설정

중요:
- 컬럼명도 버튼명도 모두 “설정”으로 통일해주세요
- 사용자가 봤을 때 이 버튼이 단순 조회가 아니라
  직원별 운영 기준을 설정하는 액션으로 읽혀야 합니다

즉,
- 테이블 헤더: 관리 → 설정
- 각 행 버튼: 관리 → 설정

[2. 조 배치 결과를 테이블에 표시]
현재 테이블에서는 역할이
- 인차지
- 담당
처럼 태그/칩으로 표시되고 있는데,
직원별 조 배치도 같은 방식으로 메인 테이블에 보이게 수정해주세요.

즉,
설정 버튼을 눌러 직원별 조 배치를 지정하면
그 결과가 메인 테이블에 바로 표시되어야 합니다.

표시 대상 예시:
- 오전조
- 오후조
- 야간조
- 중간조

중요:
- 역할이 태그형으로 보이는 것처럼
- 조 배치도 같은 디자인 체계의 태그/칩 형태로 보여야 합니다
- 사용자가 테이블만 봐도
  이 직원이 현재 어떤 조로 운영 배치되어 있는지 한눈에 알 수 있어야 합니다

[3. 조 배치 컬럼 추가 또는 명확화]
메인 테이블에 아래 중 하나 방식으로 반영해주세요.

권장 방식:
- 역할 옆 또는 역할 다음에 “조 배치” 컬럼 추가
- 그 컬럼 안에 현재 배정된 조를 태그 형태로 표시

예시:
- 역할: 인차지
- 조 배치: 오전조

또는
- 역할: 담당
- 조 배치: 중간조

중요:
- 현재 “역할”만 보이는 구조로는 부족
- 역할과 조 배치가 별도로 구분되어 보여야 함
- 두 값이 섞여 보이지 않게 명확히 분리

[4. 설정 버튼과 조 배치 연동]
오른쪽 “설정” 버튼을 누르면 열리는 팝업/패널에서
직원의 조 배치를 지정할 수 있어야 하고,
저장 후 그 결과가 메인 테이블의 “조 배치” 영역에 바로 반영되게 해주세요.

즉, 흐름은 아래처럼 보이게 해주세요.
1. 직원 행의 “설정” 버튼 클릭
2. 설정 팝업 또는 상세 패널 열림
3. 역할(인차지/담당) 설정 가능
4. 조 배치(오전조/오후조/야간조/중간조) 설정 가능
5. 저장
6. 메인 테이블에 역할과 조 배치가 각각 반영

중요:
- 역할과 조 배치는 모두 설정 화면에서 관리
- 저장 후 메인 테이블에 결과가 즉시 보이게

[5. 조 배치 표시 디자인]
조 배치 표시는 역할 태그와 같은 컴포넌트 체계 안에 있는 것처럼 보여야 합니다.

예:
- 역할 태그: 인차지 / 담당
- 조 배치 태그: 오전조 / 오후조 / 야간조 / 중간조

디자인 방향:
- 동일한 칩/태그 계열
- 역할과는 구분되지만 같은 시스템 안에 있는 느낌
- 너무 화려하지 않게
- 기존 근무코드 범례와도 자연스럽게 연결되는 톤

권장:
- 조별로 미세하게 다른 톤 사용 가능
- 하지만 과한 색상 대비는 피하고 관리자용 UI답게 정돈되게 유지

[6. 메인 테이블에서 읽혀야 하는 정보]
직원 관리 테이블은 아래를 한눈에 보여주는 구조여야 합니다.

- 이름
- 직급
- 역할
- 조 배치
- 전체 근속
- 프런트 시작일
- 현재 프런트 근속
- 누적 프런트 경력
- 상태
- 설정

즉,
이 직원이
- 어떤 직급인지
- 어떤 역할인지
- 현재 어느 조로 운영 배치되는지
를 테이블 수준에서 바로 확인할 수 있어야 합니다.

[7. 최종 요청]
현재 직원 관리 화면을 아래 기준으로 수정해주세요.

- 제일 오른쪽 액션 버튼명 “관리”를 “설정”으로 변경
- 관련 컬럼명도 “설정”으로 통일
- 직원별 조 배치 결과를 메인 테이블에 표시
- 역할이 인차지/담당으로 보이는 것처럼
  조 배치도 오전조/오후조/야간조/중간조 태그로 보이게 수정
- 설정 버튼을 눌러 조 배치를 지정하면 저장 후 테이블에 즉시 반영되게 정리
- 역할과 조 배치는 별도 컬럼 또는 명확히 구분된 정보로 보이게 수정

핵심은,
직원 관리 화면에서
“역할”뿐 아니라 “현재 조 배치”까지 테이블에서 바로 읽히도록 만드는 것입니다.

핵심: 오른쪽 버튼명은 “설정”으로 바꾸고, 역할 태그처럼 직원별 조 배치(오전조/오후조/야간조/중간조)도 메인 테이블에 바로 보이게 해주세요.
