현재 LOTTE HOTELS & RESORTS 관리자용 SaaS의 “직원 관리” 화면을 아래 기준으로 수정해주세요.

이번 수정은 직원별 역할 선택 위치를 바꾸고, 불필요한 조 관련 기능을 제거하며, 우측 액션 버튼의 의미를 더 명확하게 정리하는 작업입니다.

[1. 왼쪽 “관리” 컬럼 삭제]
현재 테이블 가장 왼쪽에 있는
- 인차지 / 담당 드롭다운이 들어간 “관리” 컬럼은 삭제해주세요.

중요:
- 더 이상 메인 테이블 왼쪽에서 바로 역할을 선택하지 않게 해주세요.
- 즉, 현재 왼쪽 관리 컬럼 전체를 제거해주세요.

[2. 오른쪽 “배치” 버튼명 변경]
현재 테이블 가장 오른쪽에 있는 “배치” 버튼의 이름을 아래처럼 변경해주세요.

변경 전:
- 배치

변경 후:
- 관리

즉, 우측 액션 버튼은 모두 “관리”로 통일해주세요.

[3. 역할 선택 위치 변경]
기존에 왼쪽 드롭다운에서 하던
- 인차지
- 담당

선택 기능을
우측 “관리” 버튼을 눌렀을 때 열리는 팝업 / 탭 / 상세 패널 안으로 이동해주세요.

즉, 흐름은 아래처럼 바꿔주세요.

새 흐름:
1. 사용자가 직원 행의 오른쪽 “관리” 버튼 클릭
2. 직원별 관리 팝업 또는 우측 상세 패널 오픈
3. 그 안에서 아래 항목을 설정
   - 역할: 인차지 / 담당
   - 기타 직원 운영 관련 설정
4. 저장 또는 적용
5. 결과가 메인 테이블에 반영

중요:
- 역할 선택은 더 이상 테이블 본문에서 직접 하지 않음
- 반드시 “관리” 액션 안에서 설정하는 방식으로 정리해주세요.

[4. “배치 가능한 조” 기능 완전 삭제]
“배치 가능한 조”는 필요 없는 기능이므로 전부 삭제해주세요.

삭제 이유:
모든 직원은 기본적으로 모든 조에 들어갈 수 있기 때문에
직원별로 “배치 가능한 조”를 설정하거나 보여줄 필요가 없습니다.

삭제 대상:
- 직원 관리 테이블 안의 “배치 가능한 조” 관련 컬럼
- 직원 관리 팝업 / 상세 패널 안의 “배치 가능한 조” 설정 영역
- 직원 스케줄 배치 팝업 안의 “배치 가능한 조” 관련 선택 UI
- 관련 설명 문구, 태그, 라벨, 필터도 있으면 모두 제거

중요:
- 표에서도 삭제
- 직원 스케줄 배치하는 팝업에서도 삭제
- 이 기능은 화면 전체에서 완전히 빠져야 함

[5. 관리 팝업에서 남길 핵심 기능]
우측 “관리” 버튼을 눌렀을 때 열리는 화면에는 아래 내용이 자연스럽게 포함되게 해주세요.

필수 포함 항목:
- 직원명
- 직급
- 역할 선택: 인차지 / 담당
- 입사일
- 전체 근속
- 프런트 시작일
- 현재 프런트 근속
- 누적 프런트 경력
- 복귀자 여부 및 관련 경력 정보(기존 설정 유지 시)
- 저장 / 취소 버튼

즉, 이 팝업은 단순 “배치”가 아니라
직원별 운영 기준을 설정하는 “관리 팝업”처럼 보여야 합니다.

[6. 메인 테이블 정리 방향]
수정 후 메인 테이블은 더 간결하게 정리해주세요.

유지할 정보 예시:
- 이름
- 직급
- 역할
- 전체 근속
- 프런트 시작일
- 현재 프런트 근속
- 누적 프런트 경력
- 상태
- 관리 버튼

제거할 것:
- 왼쪽 관리 컬럼
- 배치 가능한 조 관련 정보

즉,
테이블은 “결과 확인용”
우측 관리 버튼은 “설정/수정용”
구조로 정리해주세요.

[7. 역할 반영 방식]
관리 팝업에서 선택한 역할 값은
저장 후 메인 테이블의 역할 영역에 즉시 반영되어 보여야 합니다.

예:
- 관리 팝업에서 인차지 선택 → 테이블에 인차지 표시
- 관리 팝업에서 담당 선택 → 테이블에 담당 표시

표현 방식:
- 기존 시스템의 태그/칩 스타일 유지
- 관리자용 실무 UI답게 단정하게 보이게

[8. 전체 확정 흐름은 유지]
기존에 논의한 “전체 확정” 기능은 유지해주세요.

즉,
- 직원별 관리 버튼으로 역할 등 정보 수정
- 모든 직원 설정 완료
- 전체 확정 버튼 클릭
- 근무표 생성 기준으로 일괄 반영

이 흐름은 그대로 유지하되,
왼쪽 즉시 선택형 UI만 제거하고
오른쪽 관리 버튼 중심 구조로 정리해주세요.

[9. 톤앤매너 유지]
반드시 현재까지 만든 LOTTE HOTELS & RESORTS 관리자용 SaaS의 톤앤매너를 유지해주세요.

유지 조건:
- 프리미엄 호텔 운영 관리자용 SaaS 무드
- 딥 네이비 / 웜 화이트 / 절제된 골드 포인트
- 얇은 선, 넓은 여백, 정돈된 그리드
- 과한 HR 시스템 느낌 금지
- 소비자용 앱 느낌 금지
- 운영팀이 실제 쓰는 실무형 관리자 화면처럼 보이게

[10. 최종 요청]
현재 직원 관리 화면을 아래 기준으로 수정해주세요.

- 왼쪽 인차지/담당 드롭다운이 있는 “관리” 컬럼 삭제
- 오른쪽 “배치” 버튼명을 “관리”로 변경
- 역할 선택(인차지 / 담당)은 오른쪽 관리 버튼을 눌렀을 때 열리는 팝업/탭 안에서 하도록 이동
- “배치 가능한 조” 기능은 표에서도 삭제
- 직원 스케줄 배치 팝업에서도 “배치 가능한 조” 기능 삭제
- 메인 테이블은 더 간결하게 정리
- 역할 설정 결과는 테이블에 반영
- 전체 확정 흐름은 유지

이 기준으로 UI와 기능 흐름을 다시 정리해주세요.

중요: 테이블은 결과 확인용으로 단순화하고, 직원별 설정은 모두 오른쪽 “관리” 액션 안에서 하도록 구조를 바꿔주세요.
