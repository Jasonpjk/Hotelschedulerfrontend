현재 LOTTE HOTELS & RESORTS 관리자용 SaaS의 “근무표” 화면을 아래 기준으로 수정해주세요.

중요:
- 이번 수정은 “근무표” 화면에만 적용해주세요.
- 다른 화면은 건드리지 말아주세요.
- 기존 시스템의 프리미엄 호텔 운영 관리자용 SaaS 톤앤매너 유지
- 딥 네이비 / 웜 화이트 / 절제된 골드 포인트 / 넓은 여백 / 얇은 선 / 정돈된 그리드 유지

[현재 핵심 문제]
현재 근무표 화면에서 아래 2가지 문제가 있습니다.

1) AI 자연어 조정 기능 문제
- AI 자연어 조정 입력창에 명령을 입력하고 “적용”을 눌러도
  하단 “변경 이력”에 자동으로 기록이 추가되지 않음
- 원래 있어야 하는 변경 이력 자동 누적 기능이 사라짐

2) 메인 스케줄 달력 가시성 문제
- 근무표의 메인 스케줄 달력/그리드가 이 화면의 핵심인데
  현재 스크롤 구조나 레이아웃 때문에 가려져서 거의 보이지 않음
- 사용자가 가장 먼저 봐야 할 스케줄 달력이 메인처럼 보이지 않음

[1. AI 자연어 조정 적용 시 변경 이력 자동 추가 기능 복구]
AI 자연어 조정 영역의 원래 기능을 다시 복구해주세요.

원하는 흐름:
1. 사용자가 AI 자연어 조정 입력창에 명령 입력
2. “적용” 버튼 클릭
3. 조정 적용 피드백 표시
4. 화면 하단 “변경 이력”에 새 이력이 자동으로 1건 추가

즉,
AI 조정은 단순 입력 후 끝나는 기능이 아니라
무슨 명령이 언제 적용됐는지 하단에 기록이 남아야 합니다.

[2. 변경 이력에 반드시 들어갈 정보]
하단 변경 이력에는 아래 정보가 자동으로 추가되게 해주세요.

필수 항목:
- 적용 시각
- 입력한 자연어 명령
- 반영 상태
- 적용 주체 또는 유형: AI 조정

예시:
- 2026-04-03 14:32 / “주말 OFF를 더 균등하게 조정” / 적용 완료
- 2026-04-03 14:40 / “4월 12일 야간조 커버리지 보완” / 일부 반영
- 2026-04-03 14:48 / “인차지 배치를 오전조에 우선 보강” / 적용 완료

중요:
- 적용 버튼을 누르면 흔적이 반드시 남아야 함
- 변경 이력이 자동 누적되어야 함
- 기존 이력은 유지되고, 새 이력이 추가되는 구조여야 함

[3. 변경 이력 섹션 복구/강화]
근무표 하단의 “변경 이력” 또는 “AI 조정 이력” 섹션이
실제로 작동하는 영역처럼 다시 보이게 해주세요.

구성 방향:
- AI 자연어 조정 입력창 바로 아래 또는 하단 로그 영역
- 리스트형 또는 테이블형
- 최신 이력이 누적
- 최신순 정렬 가능
- 상태 태그 예:
  - 적용 완료
  - 일부 반영
  - 검토 필요

중요:
- 채팅창처럼 보이면 안 됨
- 근무표 작업 이력/로그처럼 보여야 함

[4. 메인 스케줄 달력/그리드가 화면의 중심이 되게 레이아웃 수정]
현재 근무표 화면에서 메인 스케줄 달력/그리드가 스크롤에 가려져 잘 안 보이는 문제를 수정해주세요.

수정 방향:
- 스케줄 달력/그리드가 화면에서 가장 넓고 가장 중심적인 영역이 되게
- 주변 패널이나 하단 요소 때문에 메인 표가 눌리거나 가려지지 않게
- 사용자가 화면에 들어왔을 때 가장 먼저 스케줄 표가 보여야 함

즉,
근무표 화면은 다른 보조 요소보다
스케줄 그리드가 메인이라는 점이 분명해야 합니다.

[5. 스크롤 구조 재정리]
현재는 스크롤 구조 때문에 근무표가 가려지거나
중요한 메인 표가 충분히 보이지 않는 상태이므로
스크롤 방식을 다시 정리해주세요.

권장 방향:
- 전체 페이지 스크롤과 표 내부 스크롤 역할 분리
- 상단 요약 / 범례 / 조정 영역은 과도하게 공간 차지하지 않게
- 메인 표는 충분한 높이를 확보
- 표가 메인 워크스페이스처럼 보이게

중요:
- 메인 스케줄 표가 잘려 보이거나 하단으로 밀려나면 안 됨
- 사용자가 불필요하게 많이 스크롤해야 스케줄이 보이는 구조는 피해야 함

[6. 우선순위 정리]
근무표 화면에서 보여지는 우선순위는 아래처럼 정리해주세요.

1순위:
- 메인 스케줄 달력/그리드

2순위:
- 상단 상태/버전/확정 관련 컨트롤

3순위:
- AI 자연어 조정 입력창

4순위:
- 변경 이력

즉,
변경 이력은 중요하지만 메인 스케줄 표를 가릴 정도로 크면 안 되고,
AI 자연어 조정도 보조 도구이지 메인 표를 압도하면 안 됩니다.

[7. AI 조정 영역과 변경 이력 영역 크기 조정]
AI 자연어 조정 입력 영역과 하단 변경 이력 영역은
스케줄 표를 침범하지 않는 선에서 정돈되게 조정해주세요.

권장 방향:
- 입력창은 한 줄 또는 컴팩트한 높이 유지
- 변경 이력은 필요 이상으로 높지 않게
- 필요 시 접기/펼치기 또는 스크롤 가능한 로그 영역처럼 보이게 가능
- 하지만 메인 표가 충분히 먼저 보여야 함

[8. 근무표 화면의 최종 인상]
사용자가 이 화면을 봤을 때 아래처럼 느껴져야 합니다.

- 이 화면의 주인공은 스케줄 달력/표다
- AI 자연어 조정은 메인 표를 보조하는 도구다
- 적용한 AI 명령은 아래 변경 이력에 자동으로 남는다
- 작업 과정이 추적 가능하다

[9. 최종 요청]
현재 근무표 화면을 아래 기준으로 수정해주세요.

- AI 자연어 조정에 명령 입력 후 “적용”을 누르면 하단 변경 이력에 자동으로 새 기록이 추가되는 기능 복구
- 변경 이력에는 적용 시각, 입력 명령, 반영 상태가 포함되게
- 메인 스케줄 달력/그리드가 스크롤에 가려지지 않게 레이아웃 수정
- 근무표 화면에서 스케줄 표가 가장 넓고 중심적인 메인 영역으로 보이게 정리
- AI 조정 영역과 변경 이력 영역은 보조 영역으로 정리하여 메인 표를 침범하지 않게 조정

핵심은,
사라진 “AI 조정 후 변경 이력 자동 누적” 기능을 복구하고,
근무표의 메인 스케줄 달력이 화면에서 가장 잘 보이도록 레이아웃을 바로잡는 것입니다.

중요: 근무표 화면에서 가장 먼저 보여야 하는 것은 스케줄 달력/그리드이며, AI 조정 적용 시 하단 변경 이력 자동 추가 기능은 반드시 다시 살아나야 합니다.
