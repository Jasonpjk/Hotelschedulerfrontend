현재 LOTTE HOTELS & RESORTS 관리자용 SaaS의 “근무표” 화면을 아래 기준으로 수정해주세요.

중요:
- 이번 수정은 “근무표” 화면에만 적용해주세요.
- 다른 화면은 건드리지 말아주세요.
- 기존 시스템의 프리미엄 호텔 운영 관리자용 SaaS 톤앤매너 유지
- 딥 네이비 / 웜 화이트 / 절제된 골드 포인트 / 넓은 여백 / 얇은 선 / 정돈된 그리드 유지

[1. 월 변경 시 버전명도 해당 월 기준으로 바뀌어야 함]
현재 근무표 화면에서 대상 월을 바꾸면,
옆에 보이는 버전명도 그 월에 맞는 버전으로 바뀌게 수정해주세요.

예시:
- 2026년 3월 선택 시: v3.1
- 2026년 4월 선택 시: v4.0

즉,
버전은 화면에 고정된 하나의 값이 아니라
“선택한 월의 근무표 버전”으로 연동되어야 합니다.

중요:
- 사용자가 월 드롭다운을 변경하면
  버전 드롭다운 또는 현재 버전 표시도 즉시 그 월 기준으로 전환되어야 함
- 3월 화면의 버전을 그대로 들고 가면 안 됨
- 월별로 독립된 버전 체계를 가진 것처럼 보여야 함

[2. 변경 이력도 월별로 분리]
근무표 하단의 변경 이력은
현재 선택한 월의 이력만 보여야 합니다.

즉,
- 2026년 3월 선택 시 → 3월 변경 이력만 표시
- 2026년 4월 선택 시 → 4월 변경 이력만 표시
- 3월에서 4월로 바꾸면 3월 변경 이력은 화면에서 사라져야 함

중요:
- 변경 이력은 전월/타월 이력이 섞이면 안 됩니다
- 현재 선택한 월의 작업 로그만 보이게 정리해주세요
- 사용자가 “지금 보고 있는 월의 작업 이력”만 읽고 있다고 느껴야 합니다

[3. 월 전환 시 화면 전체 컨텍스트가 바뀌는 느낌 필요]
월을 변경하면 아래 요소가 함께 해당 월 기준으로 바뀌는 흐름이 느껴져야 합니다.

반드시 연동되어 보일 요소:
- 현재 월
- 현재 버전명
- 근무표 데이터
- 경고 요약
- 하단 변경 이력

즉,
월 드롭다운을 바꾸는 순간
단순 제목만 바뀌는 것이 아니라
“해당 월의 근무표 작업 공간으로 전환되었다”는 느낌이 분명해야 합니다.

[4. 버전명 표시 방식]
버전명은 현재 선택한 월 기준으로 자연스럽게 보여야 합니다.

예시:
- 2026년 3월 + v3.1
- 2026년 4월 + v4.0

중요:
- 버전명이 독립값처럼 보이지 말고
- 선택 월에 종속된 버전값처럼 읽혀야 합니다
- 월이 바뀌면 버전도 같이 바뀌는 것이 자연스럽게 보여야 함

[5. 변경 이력 영역 제목/설명도 월 기준으로 읽히게]
하단 변경 이력 영역이 있다면,
현재 선택한 월의 이력이라는 점이 자연스럽게 드러나게 해주세요.

예:
- 변경 이력
- 2026년 4월 작업 이력
- 현재 월 기준 변경 이력

꼭 제목을 바꿀 필요는 없지만,
최소한 사용자가 혼동하지 않게
현재 월의 로그만 보인다는 구조가 느껴져야 합니다.

[6. 월 변경 예시를 화면에 자연스럽게 반영]
이번 수정의 핵심 예시는 아래입니다.

반드시 이런 동작처럼 보여야 합니다:
- 2026년 3월 화면에서 v3.1 표시
- 2026년 4월로 월 변경
- 버전명이 v4.0으로 변경
- 3월 변경 이력은 사라짐
- 4월 변경 이력만 표시

즉,
월별로 버전과 이력이 분리된 구조가 화면에서 명확히 보여야 합니다.

[7. 최종 요청]
현재 근무표 화면을 아래 기준으로 수정해주세요.

- 월을 변경하면 옆의 버전명도 해당 월 기준으로 자동 변경
- 예: 2026년 4월 선택 시 v4.0으로 변경
- 하단 변경 이력은 현재 선택한 월의 이력만 표시
- 3월에서 4월로 바꾸면 3월 변경 이력은 사라지고 4월 변경 이력만 보여야 함
- 월 / 버전 / 근무표 / 경고 / 변경 이력이 모두 같은 월 컨텍스트로 움직이게 정리

핵심은,
근무표 화면이 월별로 독립된 작업 공간처럼 보이도록
버전과 변경 이력이 현재 선택 월에 정확히 연동되어야 한다는 점입니다.

핵심: 월을 바꾸면 버전명과 하단 변경 이력도 반드시 그 월 기준으로 함께 바뀌어야 하며, 이전 월 이력은 화면에서 보이지 않게 해주세요.
