현재 LOTTE HOTELS & RESORTS 관리자용 SaaS의 “근무표” 화면을 아래 기준으로 수정해주세요.

중요:
- 이번 수정은 “근무표” 화면에만 적용해주세요.
- 다른 화면은 건드리지 말아주세요.
- 기존 시스템의 프리미엄 호텔 운영 관리자용 SaaS 톤앤매너 유지
- 딥 네이비 / 웜 화이트 / 절제된 골드 포인트 / 넓은 여백 / 얇은 선 / 정돈된 그리드 유지

[1. 확정 버튼 클릭 시 상태값 변경]
현재 상단 버전 옆 상태값이 “작업 중”으로 보이는데,
확정 버튼을 누르면 이 상태값이 반드시 “확정”으로 변경되게 해주세요.

즉, 흐름은 아래처럼 보이게 해주세요.
- 확정 전: 작업 중
- 확정 버튼 클릭 후: 확정

중요:
- 단순히 버튼만 눌리는 것이 아니라
- 현재 버전 상태가 실제로 “확정”된 것으로 읽혀야 합니다
- 상단 상태 뱃지 또는 상태 텍스트가 명확히 바뀌어야 합니다

[2. 새 버전 생성 시 버전 드롭다운에 새 버전 추가]
현재 “새 버전 생성” 버튼을 누르면
왼쪽 버전 드롭다운에 새로운 버전이 자동으로 추가되게 해주세요.

원하는 흐름:
1. 사용자가 “새 버전 생성” 버튼 클릭
2. 새 버전 생성
   예: v3.2 또는 다음 버전 번호
3. 버전 드롭다운 목록에 자동 추가
4. 새 버전이 자동 선택됨
5. 새 버전 옆 상태는 “작업 중”으로 표시

즉,
새 버전 생성 후에는
- 기존 확정 버전은 그대로 유지
- 새 버전이 드롭다운에 추가
- 새 버전은 수정 가능한 작업 버전처럼 보여야 합니다

중요:
- 새 버전 생성은 기존 버전을 덮어쓰는 것이 아님
- 새로운 작업용 버전을 분기하는 흐름처럼 보여야 함

[3. 새 버전 상태값 표시]
새 버전 생성 후에는
왼쪽 버전 드롭다운 옆 또는 상태 영역에 반드시 “작업 중” 상태가 보여야 합니다.

예:
- v3.1 + 확정
- 새 버전 생성
- v3.2 + 작업 중

중요:
- 사용자가 현재 선택한 버전이 확정본인지 작업본인지 바로 알 수 있어야 함
- 새 버전은 기본적으로 “작업 중” 상태여야 합니다

[4. 확정 후 셀 수동 조정 기능 비활성화]
현재 근무표에서는 각 스케줄 셀을 클릭해서 수동 조정 팝업을 띄울 수 있는데,
확정 버튼을 누른 후에는 이 기능이 비활성화되어야 합니다.

즉, 흐름은 아래처럼 보이게 해주세요.
- 작업 중 상태: 셀 클릭 가능 / 수동 조정 가능
- 확정 상태: 셀 클릭 불가 / 수동 조정 팝업 열리지 않음

중요:
- 확정된 버전에서는 더 이상 직접 셀 편집이 되면 안 됨
- 사용자가 셀을 눌렀을 때 수정 가능한 것처럼 보이지 않게 해주세요
- 필요하면 잠금 상태가 느껴지도록 인터랙션을 정리해주세요

[5. 확정된 셀 인터랙션 정리]
확정 상태가 되면 각 셀은 아래처럼 보여야 합니다.
- 클릭해도 팝업이 열리지 않음
- 손가락 커서처럼 보이지 않음
- 수정 가능한 셀처럼 hover 강조되지 않음
- 필요하면 “확정된 버전은 수정할 수 없습니다” 같은 보조 안내 가능

중요:
- 확정 후에는 사용자가 자연스럽게
  “이제 이 버전은 잠겨 있구나”라고 이해할 수 있어야 합니다

[6. 전체 상태 흐름이 분명하게 보이게]
근무표 화면의 버전 흐름은 아래처럼 읽히게 정리해주세요.

권장 흐름:
1. 기존 버전 선택
2. 상태: 작업 중
3. 셀 수정 가능
4. 확정 버튼 클릭
5. 상태: 확정
6. 셀 수정 불가
7. 새 버전 생성
8. 새 버전이 드롭다운에 추가
9. 새 버전 상태: 작업 중
10. 새 버전에서 다시 셀 수정 가능

즉,
“작업 중 → 확정 → 새 버전 생성 → 다시 작업 중”
흐름이 명확히 보여야 합니다.

[7. 최종 요청]
현재 근무표 화면을 아래 기준으로 수정해주세요.

- 확정 버튼을 누르면 상단 상태값이 “작업 중”에서 “확정”으로 변경
- 새 버전 생성을 누르면 왼쪽 버전 드롭다운에 새로운 버전이 자동 추가
- 새 버전이 자동 선택되고 옆에 “작업 중” 상태가 보이게 구성
- 확정 버튼을 누르면 스케줄 각 셀 클릭을 통한 수동 조정 기능이 비활성화
- 확정된 버전은 수정 불가, 새 버전은 수정 가능이라는 흐름이 화면에서 명확히 보이게 정리

핵심은,
확정과 새 버전 생성이 각각
“버전 상태 변경”과 “편집 가능 여부 변경”으로 분명히 연결되어 보이도록 만드는 것입니다.
핵심: 확정 시 상태는 “확정”으로 바뀌고 셀 편집은 잠기며, 새 버전 생성 시 드롭다운에 새 버전이 추가되고 그 버전은 “작업 중” 상태로 다시 편집 가능해야 합니다.
