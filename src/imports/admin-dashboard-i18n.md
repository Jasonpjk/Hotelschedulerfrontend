현재 대시보드의 디자인, 레이아웃, 컬러, 분위기는 그대로 유지하되,
다국어 호환이 완전히 되는 관리자 대시보드로 수정해줘.

현재 상태 규칙:
- 지금 화면은 일본어로 로그인한 상태다
- 따라서 브랜드명, 호텔명, 사람 이름 같은 고유명사를 제외한 모든 기능 UI 텍스트는 100% 일본어로 표시되어야 한다
- 영어, 한국어, 일본어가 한 화면 안에 섞이면 안 된다
- 현재 화면에 남아 있는 영어 UI 이름을 모두 일본어로 바꿔줘

다국어 호환 규칙:
- 이 대시보드는 한국어, 영어, 일본어, 베트남어, 러시아어를 모두 지원하는 구조여야 한다
- 각 언어 선택 시, 기능 UI 텍스트는 해당 언어 하나로만 100% 통일되어야 한다
- 언어 혼용 금지
- 텍스트 길이가 가장 긴 언어도 수용할 수 있도록 설계해줘
- 버튼 폭, 카드 폭, 사이드바 너비, 헤더 영역, 표 제목, 상태 배지, 링크 텍스트가 길어져도 줄바꿈이나 잘림이 어색하지 않게 해줘
- 일본어는 짧고 조밀한 구조
- 영어는 중간 길이
- 베트남어와 러시아어는 더 길어질 수 있으므로 여백과 폭을 충분히 고려
- 한국어도 제목이 길어질 수 있으니 안정적인 레이아웃 유지
- 언어별로 글꼴이 바뀌더라도 전체 톤앤매너는 동일해야 한다

타이포그래피 규칙:
- 정확히 특정 폰트를 복제하기보다, 프리미엄 호텔 브랜드 느낌의 고급스럽고 정제된 타이포그래피 무드를 유지
- 한글, 영어, 일본어, 베트남어, 러시아어 모두 자연스럽게 보이는 국제화 친화적 구조
- 제목은 고급스럽고 절제된 인상
- 기능 텍스트와 데이터 텍스트는 높은 가독성 우선
- 언어가 바뀌어도 시각적 위계가 무너지지 않게 해줘

디자인/브랜드 조건:
- 로그인 페이지와 연결된 프리미엄 호텔 브랜드 무드 유지
- 롯데호텔 홈페이지가 주는 고급스럽고 정제된 느낌 참고
- 소비자용 예약 사이트처럼 보이지 않게
- 관리자용 운영 대시보드 느낌 유지
- 디자인은 유지하고, 언어 시스템과 UI 텍스트 일관성, 국제화 대응 구조를 강화해줘

최종 목표:
- 현재 일본어 선택 상태에서는 기능 UI가 모두 일본어로 통일된 대시보드
- 동시에 한국어, 영어, 일본어, 베트남어, 러시아어로 전환해도 레이아웃이 안정적으로 유지되는 국제화 대응 관리자 화면
